Windows Terminal 主题美化

发表于2021-11-24|更新于2021-12-01|Windows
|字数总计:653|阅读时长:2分钟|阅读量:|评论数:

安装 Windows Terminal

如果你是 Windows 10 系统，可以打开系统自带的 Microsoft Store，搜索 Windows Terminal 直接安装就可以了。

安装PowerShell

PowerShell软件介绍

PowerShell软件下载

PowerShell安装步骤（略）

安装Nerd Fonts

Nerd Fonts字体介绍

下载一款字体（推荐JetBrainsMono Nerd Font）

Nerd Font安装

注意：一定要安装一款字体，否则主题安装后符号或字体会乱码

  1. 解压JetBrainsMono.zip，

  2. 文件夹下的字体全部选中，点击右键

  3. 点击菜单上的安装

Oh-My-Posh、Posh-Git安装

在powershell中输入以下命令，安装oh-my-posh、Posh-Git

Install-Module oh-my-posh -Scope CurrentUser

Install-Module posh-git -Scope CurrentUser

PowerShell设置主题

  • 在powershell中输入以下命令

    notepad $PROFILE

  • 在打开的txt文本中，输入以下命令，然后保存。

    Import-Module oh-my-posh
Import-Module posh-git
Set-PoshPrompt -Theme paradox

使用 ColorTool 自定义 PowerShell 文字颜色

ColorTool 是微软出品的更改 PowerShell 文字颜色的工具，使用也非常简单。

# 安装微软官方颜色工具
> scoop install colortool

# 查看内置的配色方案，共有 8 种
> colortool --schemes

# 设置主题，后面是配色方案名称。
> colortool OneHalfDark.itermcolors

windows terminal中管理powershell

  • 打开windows terminal，打开设置

  • 设置启动默认配置文件，选择刚安装的powershell

  • 设置powershell启动时，不输出日志命令行最后面添加 -nologo

  • 设置powershell的字体，选择刚装过的JetBrainsMono字体

  • 最后保存，在windows terminal中查看美化后的powershell吧

查看Oh-My-Posh主题

在powershell中输入以下命令：

Get-PoshThemes

PowerShell修改主题

输入notepad $PROFILE

PowerShell功能增强

增强 PowerShell 的 ls 功能

dircolors 是 Linux 下的命令，可以设置 ls 指令在显示目录或文件时的色彩，同样的，如果也想让 PowerShell 显示彩色目录，可用插件 DirColors 实现。

# 安装 DirColors
> Install-Module DirColors
# 打开 PowerShell 配置
> notepad $PROFILE

在配置文件末尾添加一行，重启 PowerShell 就可以看到彩色目录了。

Import-Module DirColors

补全功能

  • 根据历史记录补全，Tab 补全，菜单补全 & Intellisense等等
# 打开 PowerShell 配置
> notepad $PROFILE

# 在配置文件末尾添加
Set-PSReadLineOption -PredictionSource History # 设置预测文本来源为历史记录
 
Set-PSReadlineKeyHandler -Key Tab -Function Complete # 设置 Tab 键补全
Set-PSReadLineKeyHandler -Key "Ctrl+d" -Function MenuComplete # 设置 Ctrl+d 为菜单补全和 Intellisense
Set-PSReadLineKeyHandler -Key "Ctrl+z" -Function Undo # 设置 Ctrl+z 为撤销

其他插件也可以用同样的方法安装，开始探索吧 ~
